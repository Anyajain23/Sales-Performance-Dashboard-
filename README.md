<h1 align="center">📊 Sales Performance Dashboard</h1>

<p align="center">
Power BI Data Analysis Project
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project presents a <b>Sales Performance Dashboard</b> created using <b>Microsoft Power BI</b>.
The dashboard analyzes sales data to track revenue, profit, and performance across different regions and product categories.
</p>

<p>
The project also includes a <b>15-day sales forecasting feature</b> to predict future sales trends based on historical data.
</p>

<hr>

<h2>📊 Dataset Information</h2>

<ul>
<li>Order Date</li>
<li>Customer Segment</li>
<li>Region</li>
<li>Product Category</li>
<li>Sales Amount</li>
<li>Profit</li>
<li>Quantity Sold</li>
</ul>

<p>
The dataset contains more than <b>10,000+ sales records</b> used for sales analysis and forecasting.
</p>

<hr>

<h2>📈 Dashboard Features</h2>

<ul>
<li>Total Sales KPI</li>
<li>Total Profit KPI</li>
<li>Sales Trend Analysis</li>
<li>Sales by Region</li>
<li>Sales by Product Category</li>
<li>Top Performing Products</li>
<li>Interactive Filters and Slicers</li>
<li><b>15-Day Sales Forecasting using Power BI Time Series Forecast</b></li>
</ul>

<hr>

<h2>🛠 Tools Used</h2>

<ul>
<li>Microsoft Power BI</li>
<li>Power Query for Data Cleaning</li>
<li>DAX for Measures</li>
<li>Data Visualization</li>
</ul>

<hr>

<h2>🔍 Key Insights</h2>

<ul>
<li>Sales performance varies significantly across different regions.</li>
<li>Certain product categories generate higher revenue.</li>
<li>Monthly sales trends highlight peak sales periods.</li>
<li>The <b>15-day forecast</b> helps estimate future sales trends and supports better business planning.</li>
</ul>


<h2>🎯 Project Objective</h2>

<ul>
<li>Analyze sales performance using Power BI</li>
<li>Create an interactive dashboard for business insights</li>
<li>Apply time-series forecasting to predict future sales</li>
<li>Build a portfolio project for data analyst roles</li>
</ul>
